# Sakok Ui

## Install

1. install package 
	1. Run `npm install https://github.com/AliAmini/sakok-antd4-component.git`

1. add component to your code
	1. import on top of your app `import Antd4Component from 'sakok-ant4-component';`
	1. use in your jsx code: 

	```
	render () {
	  return (
	    <div className="App">
	      <Antd4Component /> 
	    </div>
      );
	}
	```


## dependence 
1. node: 8.10
2. npm: 6
3. antd: 4
5. less: last
6. react: 16.8.6
